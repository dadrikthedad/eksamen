# Eksamener - Fredrik Magee

Mappeinnleveringsoppgaver og eksamensoppgaver fra Backend-programmering på Gokstad Akademiet i år 2.

Emne 5 - Algoritmiske metoder (C# konsollapplikasjon)
Emne 6 - Software design (C# Blazor WebAssembly)
Emne 7 - Asp.Net Core (C# Asp.Net Core)

Inneholder praktiske implementasjoner innen algoritmer, software design og .NET-utvikling.
Karakter fra WiseFlow er lagt med som et skjermbilde i hver oppgave og oppgaveteksten ligger med i hver besvarelse som en PDF.
